# Changelog

## [2026-07-04]

### Fixed
- 深色模式覆盖不完整，导航栏和 Hero Banner 等组件浅色残留
- SettingsPanel 滑块按钮白色背景未适配暗色模式

## [2026-07-03]

### Added
- 用户系统：注册/登录、头像上传、个人资料页、文章创作与管理
- 智能体双轨制：系统默认智能体 + 用户自建智能体
- 评论登录限制：登录后评论，自动显示头像昵称，文章作者标识
- 智能体选择器：首页侧边栏下拉切换不同智能体，对话记录独立保存
- 模型名称推荐：根据厂商自动显示推荐模型列表
- 博主信息 API 化：首页和侧边栏统一从后端获取最新数据
- 博客声明页面：默认声明"此博客服务于每一个热爱交流的技术选手"
- 文件上传功能：admin 和 user 各有独立头像上传端点

### Changed
- "关于我"页面全面改为"博客声明"
- 智能体编辑器简化：删除简介、最大Token、上下文轮数、频率限制等字段
- 温度改为"模型性格"选择框（热情/温柔/冷酷）
- 豆包模型推荐修正为实际可用模型名
- AI API 错误信息详细化：返回 HTTP 状态码和响应体
- 测试账号改为 test001/testpass001、test002/testpass002

### Fixed
- 请求拦截器 token 优先级问题：admin 和 user token 混用导致评论失败
- DataInitializer 每次重启重置 admin ID 导致 JWT 失效
- 注册时头像上传无预览

## [2026-06-05]

- 初始版本：博客系统 + AI 智能体平台
- 文章管理、分类标签、评论互动
- 多模型 AI 智能体聊天（DeepSeek / 通义千问 / 豆包 / Moonshot / 智谱GLM / OpenAI）
- Midnight Ink 设计系统
